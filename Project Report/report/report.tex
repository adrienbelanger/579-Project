\begin{abstract}
This project aims to reimplement the paper that introduced Proximal Policy Algorithms (PPO) and target three different modifications aimed at improving late-game performance on the Atari57 benchmark. \cite{PPOPaper}
The first change is to introduce value function annealing to the algorithm, as it has been shown to do better exploitation in late-game usage [CITE]. The second is to introduce a Long Short Term Memory (LSTM) model to replace the MLP architecture of the original paper and better leverage the long-term dependencies. [CITE]
Thirdly, we introduce a replay buffer to try and improve the experience usage and gathering for improving game results in the last episodes. [CITE]
\end{abstract}
